# 要件定義 (REQUIREMENTS)

## 概要
プレゼンテーション動画を簡単に作成できるバッチアプリケーション。

## 機能要件
1. **入力処理**
    - `input/` ディレクトリ内の Markdown ファイル (`.md`) と テキストファイル (`.txt`) を読み込む。
    - ファイル名の命名規則 `NNN__title` (NNNは数字) に基づき、スライドと台本を紐付ける。
    - 番号順に処理を行う。

2. **音声合成**
    - VOICEVOX を使用して、台本テキストから音声ファイル (`.wav`) を生成する。

3. **スライド動画生成**
    - Markdown の内容をスライド画像としてレンダリングする。
    - 生成された音源の長さに合わせて、静止画から無音動画 (`.nosound.mp4`) を生成する。

4. **動画結合 (個別)**
    - 音声ファイルと無音動画を結合し、各スライドの動画ファイル (`.mp4`) を生成する。

5. **動画結合 (全体)**
    - 生成された各スライドの動画を番号順に連結し、1つのプレゼンテーション動画 (`final_presentation.mp4`) を生成する。

6. **出力処理**
    - 生成物を `output/` ディレクトリに保存する。

## 非機能要件
- Windows 環境で動作すること。
- TypeScript で実装されていること。
- パッケージマネージャとして pnpm を使用すること。
- 動画処理に fluent-ffmpeg を使用すること。
